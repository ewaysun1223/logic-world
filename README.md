   ## 谬误速查

   | 说法 | 可能谬误 |
   |------|---------|
   | "大家都这么认为" | 诉诸大众 / Ad Populum |
   | "专家/权威说" | 诉诸权威 / Ad Verecundiam |
   | "他人品有问题" | 人身攻击 / Ad Hominem |
   | "没有证据证明假" | 诉诸无知 / Ad Ignorantiam |
   | "这会导致灾难" | 滑坡谬误 / Slippery Slope |
   | "要么A要么B" | 假二分 / False Dilemma |
   | "因为天然所以好" | 诉诸自然 / Appeal to Nature |
   | "你自己也这样" | 以彼之矛 / Tu Quoque |
   | "他们在隐瞒真相" | 可能涉及阴谋论思维 |
